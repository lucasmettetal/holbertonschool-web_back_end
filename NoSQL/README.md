# NoSQL

This project covers MongoDB operations using the Mongo shell and Python with PyMongo.

## Tasks

- **0-list_databases**: List all databases in MongoDB.
- **1-use_or_create_database**: Create or switch to database `my_db`.
- **3-all**: List all documents in collection `school`.
- **4-match**: List documents with `name="Holberton school"`.
- **5-count**: Count documents in collection `school`.
- **6-update**: Add `address` attribute to documents with `name="Holberton school"`.
- **7-delete**: Delete all documents with `name="Holberton school"`.
- **8-all.py**: Python function to list all documents in a collection.
- **9-insert_school.py**: Python function to insert a document in a collection.
- **10-update_topics.py**: Python function to update topics of a school document.
- **11-schools_by_topic.py**: Python function to return schools having a specific topic.
- **12-log_stats.py**: Python script displaying stats about Nginx logs stored in MongoDB.
